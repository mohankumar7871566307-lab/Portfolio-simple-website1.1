
# Personal Portfolio Website

A simple and responsive portfolio website built using HTML5 and CSS3. This project consists of two pages: a Home Page and a Contact Page. The Home Page introduces my profile, skills, and interests, while the Contact Page provides an easy way for visitors to get in touch with me.

## Features
- Clean and modern design
- Fully responsive layout
- Home page with personal introduction
- Contact page for communication
- Built with pure HTML and CSS

## Technologies Used
- HTML5
- CSS3

## Purpose
This portfolio website was created to showcase my web development skills and establish an online presence through a simple, user-friendly interface.
